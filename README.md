# Autonomous Agent for Pythagorean perpetual futures (pyth)

AA oscript:

```bash
./agent.aa
```

Test file:

```bash
./test/agent.test.oscript.js
```

## Usage

### Run test

```bash
npm run test
# or
yarn test
```

### Lint test files

```bash
npm run lint
# or
yarn lint
```
